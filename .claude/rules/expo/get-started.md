# Expo Get Started

Expo フレームワークによるクロスプラットフォームアプリ開発の導入ガイド。

## 必要要件

- Node.js (LTS 推奨) のインストール
- 対応OS: macOS、Windows (PowerShell/WSL2)、Linux

## プロジェクト作成

```bash
npx create-expo-app@latest
```

- テンプレートにはサンプルコードが含まれ、すぐに起動・動作確認が可能
- `--template` オプションで別テンプレートを指定可能

## 学習順序

1. Introduction
2. Create a project
3. Set up your environment
4. Start developing
5. Next steps

## コード生成時の注意

- Node.js のバージョン要件を確認する
- `create-expo-app` を使った初期化を推奨
- Expo Go の限界を超えた開発が必要な場合は Development Builds を案内する
